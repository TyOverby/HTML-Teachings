Links

This lesson tells you how to link to other pages.

Link to other site

<html> <body> <a href="http://google.com"> Go to Google! </a> </body> </html> Go to Google!

Link to another page in the same folder

<html> <body> <a href="page.html"> Go to page.html </a> </body> </html> Go to page.html

Using an image as a link

<html> <body> <a href="http://google.com"> <img href="http://google.com/icon.jpg" /> </a> </body> </html>
